Simple API, Powerful Results

One POST request is all you need. Here's how it works:

api_example.py 
import requests

response = requests.post(
    "https://your-domain.com/api/v1/assess",
    headers={"X-API-Key": "aa_live_your_key_here"},
    json={
        "country": "Russia",
        "device_type": "mobile",
        "hour_of_day": 3,
        "threat_score": 85,
        "distance_from_last_login": 5000
    }
)

data = response.json()
# → {"risk_score": 0.87, "action": "BLOCK", "reasons": ["high_risk_country", ...] }
